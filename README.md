# 📊 Estadísticas del YUYU

Este proyecto es un programa sencillo en **Python** que permite calcular las estadísticas de un equipo de fútbol a partir de los resultados de varios partidos.

El usuario ingresa:
- Número de partidos jugados
- Goles a favor por partido
- Goles en contra por partido

Con esta información, el programa calcula automáticamente las estadísticas generales del equipo.

---

## ⚙️ Funcionalidades

El programa calcula:

- **PJ** – Partidos Jugados  
- **PG** – Partidos Ganados  
- **PE** – Partidos Empatados  
- **PP** – Partidos Perdidos  
- **GF** – Goles a Favor  
- **GC** – Goles en Contra  
- **DG** – Diferencia de Gol  
- **Puntos Totales**

### Sistema de puntos

- 🟢 Victoria: **3 puntos**
- 🟡 Empate: **1 punto**
- 🔴 Derrota: **0 puntos**
